# Basic REST API Mahasiswa dengan Express.js

Tugas ini merupakan implementasi REST API sederhana menggunakan Node.js dan Express.js. Proyek ini mendemonstrasikan operasi CRUD (Create, Read, Update, Delete) dasar menggunakan data dummy dalam bentuk array (tanpa database).

## Fitur Utama
* Membangun server dengan Node.js dan Express.
* Implementasi HTTP Methods: GET, POST, PUT, DELETE.
* Pengolahan data dalam format JSON.
* Penggunaan URL Parameters dan Request Body.
